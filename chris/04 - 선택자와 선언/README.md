# 선택자와 선언

```html
<!doctype html>
<html>
  <head>
    <style>
      li {
        color: red;
        text-decoration: underline;
      }
    </style>
  </head>
  <body>
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </body>
</html>
```

> h1 { color: red; font-size: 12px; }

`h1` : 선택자(Selector)

`{~~}` : 선언부(Declaration)

`color`, `font-size` : 속성(Property)

`red`, `12px` : 속성값(Value)

`;` : 구분자(Declaration Separator)
